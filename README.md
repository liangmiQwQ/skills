# Liang's skills

A repo to collect my daily use skills. Including personal practices in `skills` and other imported skills.

Different from other skills repo, it is a collection repo. It would copy all personal and third-party skills listed in [sync.sh](./sync.sh) and update them with GitHub CI everyday.

It should be the only repo you need to install globally. It's opinionated and tailored to my needs. I highly suggest you to [fork it](https://github.com/liangmiQwQ/skills/fork) and clean [my-skills](./my-skills) and `SKILLS ADDING AREA` in [sync.sh](./sync.sh) if you want to use and custom it.
