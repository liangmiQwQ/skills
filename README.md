# Liang's skills

A repo to collect my daily use skills. Including personal practices in `skills` and other imported skills.

Different from other skills repo, it is a collection repo. It would copy all personal and third-party skills listed in [sync.sh](./sync.sh) and update them with GitHub CI everyday.

It should be the only repo you need to install globally. It's opinionated and tailored to my needs. I highly suggest you to [fork it](https://github.com/liangmiQwQ/skills/fork) if you want to use and custom it.

## Skills

### Installation

```sh
skills add liangmiQwQ/skills -g --yes
```

### Personal Skills

Please refer to [my-skills](./my-skills) directory to learn more. This is a collection of skills about personal habits.

I recommend you to clear it if you want to fork and custom your own skills.

### Third Party Skills

Please refer to the `SKILLS ADDING AREA` in [sync.sh](./sync.sh) to learn more. This is a collection of third-party skills, including language, framework, designing skills.

You can modify it as your personal framework / tools choose habits in your fork.
