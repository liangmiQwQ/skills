## Scene-Recommended Combinations

### Terminal Interaction Demo
```
type (0.5s) -> enter (0.5s) -> command-execute (0.5s) -> output-appear (0.6s)
```
Looping element: `cursor-blink` as idle ambient sound.

### Card Selection Flow
```
hover-subtle (0.5s, UI hover) -> click-soft (0.5s, click) -> card-snap (0.5s, position)
```
Advanced version: `card-flip` for front/back toggle.

### AI Generation Full Flow
```
generate-start (0.8s, start) -> ai-process (1.2s, processing) -> sparkle (0.8s, flash) -> complete-done (0.8s, done)
```
On error: replace `complete-done` with `error-tone`.

### Logo Reveal (Brand Moment)
```
whoosh-fast (0.6s, buildup) -> logo-reveal-v2 (1.5s, payoff) -> sparkle (0.8s, tail)
```
Simplified: `whoosh -> logo-reveal` (v7b two-piece combo).

### UI Interaction Demo (Mobile)
```
tap-finger (0.5s, tap) -> slide-in (0.6s, panel slide-in) -> toggle-on (0.5s, toggle)
```
On completion: `success-chime` or `notification-pop`.

### Data Visualization / Dashboard
```
loading-tick (0.5s, beat) x N -> complete-done (0.8s, data ready) -> achievement (1.5s, wow moment)
```

### Form Submission Flow
```
click-soft (0.5s) -> loading-tick x2 (1.0s) -> success-chime (1.0s)
```
Failure branch: `error-tone (0.7s)`.

### Magic Transform Scene
```
whoosh-fast (0.6s) -> transform (1.0s) -> sparkle (0.8s)
```
Best for: Element morphing, before/after effect comparison, "AI rewrite" demos.

## Usage Guidelines

### Volume Recommendations (from dual-track audio system)
- **SFX main track**: `1.0` (no attenuation)
- **BGM background track**: `0.4 ~ 0.5` (SFX clearly punches through)
- **Multiple SFX overlay**: Use `amix=inputs=N:duration=longest:normalize=0` to preserve dynamic range

### ffmpeg Splice Templates
```bash
# Single SFX aligned to time point:
ffmpeg -i video.mp4 -itsoffset 2.5 -i sfx/ui/click.mp3 \
  -filter_complex "[0:a][1:a]amix=inputs=2:duration=longest:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4

# Multiple SFX + BGM:
ffmpeg -i video.mp4 \
  -itsoffset 1.0 -i sfx/transition/whoosh-fast.mp3 \
  -itsoffset 1.6 -i sfx/impact/logo-reveal-v2.mp3 \
  -i bgm.mp3 \
  -filter_complex "[3:a]volume=0.4[bgm];[0:a][1:a][2:a][bgm]amix=inputs=4:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4
```

### Selection Decision Tree
1. **Tactile action** (typing/clicking/swiping) --> `keyboard/` or `ui/`
2. **Element enter/exit** --> `transition/`
3. **Container-level operation** (card/modal) --> `container/`
4. **State feedback** (success/failure/notification) --> `feedback/`
5. **Progress/time passing** --> `progress/`
6. **Brand payoff/key moment** --> `impact/`
7. **AI magic/transform** --> `magic/`
8. **Command line/code demo** --> `terminal/`

### Avoiding Audio Pileup
- Same time point: `max 2 SFX` concurrent
- When BGM is below 0.3: can have 3
- During brand impact: clear all other SFX (leave 0.2s silence before payoff)

## Prompt Writing Principles (For Reproduction)

Reference style: `apple keynote, tight, minimal, no reverb unless ambient, crisp, elegant`

**Three elements of a good prompt**:
1. **Sound physical description**: What object, what action ("mechanical keyboard single key press")
2. **Texture/style qualifier**: apple-style / ios-style / cinematic / retro
3. **Negative examples**: no reverb / clean studio / minimal

Bad: "click sound"
Good: "crisp ui button click, clean modern interface sound, apple-style, high pitched"

Bad: "magic"
Good: "bright twinkling stars sound, high pitched glittery chime, fairy dust"
