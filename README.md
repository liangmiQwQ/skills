# Liang's skills

A repo to collect my daily use skills.

Different from other skills repo, it is a collection repo. It would copy all personal and third-party skills listed in [sync.sh](./sync.sh) and update them with GitHub CI everyday.

It should be the only repo you need to install globally. I highly suggest you to [fork it](https://github.com/liangmiQwQ/skills/fork) if you want to use it.

## Skills

### Installation

```sh
skills add liangmiQwQ/skills -g --yes
```

### Personal Skills

Please refer to [my-skills](./my-skills) directory to learn more.

I recommend you to clear it if you want to fork and custom your own skills.

### Third Party Skills

Please refer to the `SKILLS ADDING AREA` in [sync.sh](./sync.sh) to learn more.
