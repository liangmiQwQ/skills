# JS/TS Library Project Tools

## Build & Bundling

- **vite-plus** with fixed extensions + dts for type declarations

## Scripts & Tooling

- **tsx** — run TypeScript files directly
- **tinyexec** — subprocess execution when needed

## Release

- **bumpp** — version bumping + git tag
- **changelogen** / **changelogithub** — changelog generation

## Package Manager

- **pnpm**
