# Case Studies Library

This directory contains curated design case studies organized by category. Each case study analyzes what makes a design effective and provides reusable patterns.

## How to Use

1. **Browse by category** — Find examples relevant to your current task
2. **Read the analysis** — Understand why the design works
3. **Extract patterns** — Identify reusable techniques and code
4. **Adapt, don't copy** — Use these as inspiration, not templates

## Contributing

To add a new case study:

1. Create a new `.md` file in the appropriate category folder
2. Follow the case study format above
3. Focus on analysis and reusable patterns, not just description
4. Include code snippets for key techniques
5. Update this README with a link to your case study

## Design Principles Across All Cases

Regardless of style or category, all featured designs share these qualities:

1. **Clear hierarchy** — You know where to look first
2. **Intentional whitespace** — Breathing room guides attention
3. **Consistent system** — Colors, fonts, spacing follow rules
4. **Emotional resonance** — The design feels appropriate for its purpose
5. **Technical excellence** — No errors, responsive, accessible

Use these case studies to understand how great designers achieve these qualities in different contexts.
